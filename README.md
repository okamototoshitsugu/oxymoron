## サイト概要

### サイトテーマ
予定を登録し。予定ごとに自己評価をし、1ヶ月毎に振り返る。

### テーマを選んだ理由
1. 人生において時間というものをとても大切にしており、特に誰と過ごしたかは、より重要だと考えているため製作。
2. スクールにてマッチングアプリやSNSサイトを製作する生徒が多いので、それ以外のものを製作したかった。
